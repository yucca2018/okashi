# okashi

### 概要
- お菓子の検索アプリです。
- API機能を使用しお菓子やアイスなどの詳細を知ることができます。

### Versions
Swift | Xcode
-|-
Swift 5.0 | Xcode12.4

### 使用フレームワーク
- SwiftUI
- UIKit
- SafariServices
- Foundation
- API

### 機能
- 検索窓から検索可能
- 該当語句からリスト表示
- リストをタップすると詳細画面へリンク
